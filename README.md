## Description
[]: {{{1

  snake.coffee - snake game library w/ bigbang.coffee

  A translation of the "snake" game from Realm of Racket [1] to
  coffeescript using the bigbang.coffee [2] library.

  To run the game, use bigbang-snake [3].
